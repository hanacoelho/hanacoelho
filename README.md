## 🎯 Visão Geral

Este projeto é um **dashboard estratégico de mentoria TikTok**, estruturado para guiar uma criadora do zero até a monetização.

---

## 🧠 Módulo 01 — Algoritmo do TikTok

O TikTok distribui conteúdo baseado em comportamento, não seguidores.

### Principais sinais:

- ⏱️ **Taxa de conclusão**  
- 🔁 **Replay**  
- 📤 **Compartilhamentos**  
- 💬 **Comentários**  
- 💾 **Salvamentos**  
- ❤️ **Curtidas**

### 🎬 Lógica de distribuição:

Seu vídeo passa por **ondas de teste**:
1. 200–500 pessoas  
2. 5.000 pessoas  
3. 50.000+  

Se performar bem → escala  
Se não → para

> 💡 Retenção é o combustível do alcance

## 🎬 Módulo 02 — Vídeos Virais

### 🪝 Hooks (3 primeiros segundos)

Exemplos:
- "Ninguém te conta isso sobre viajar barato"
- "Se você faz isso, está perdendo dinheiro"
- "Esse erro está sabotando seu treino"

---

### ⚡ Gatilhos virais

- Curiosidade  
- Surpresa  
- Identificação  
- Polêmica leve  
- Utilidade  

---

### 📣 CTA estratégico

- "Salva esse vídeo"
- "Comenta o que você acha"
- "Manda pra alguém"

## 💬 Módulo 03 — Engajamento

### Como gerar:

**Comentários**
- Perguntas diretas

**Salvamentos**
- Dicas práticas
- Tutoriais

**Compartilhamentos**
- Conteúdo identificável
- Humor + verdade

### 📖 Estrutura de Storytelling

1. Gancho (0–3s)  
2. Desenvolvimento  
3. Resolução + CTA

## 📐 Módulo 05 — Formato Ideal

### 🎥 Duração:

- 7–15s → viral rápido  
- 30–60s → melhor retenção  
- 1–3 min → storytelling  

---

### 📅 Frequência:

- 1–2 vídeos/dia (ideal início)
- mínimo: 1/dia

---

### ⏰ Horários:

- 7h–9h  
- 12h–14h  
- 19h–22h  

---

### 📊 Formatos que funcionam:

- POV  
- Storytelling  
- Lista  
- Antes/Depois  
- GRWM  
- Realidade vs Expectativa
